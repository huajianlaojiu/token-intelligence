## 📖 简介

**Token Intelligence** 是一个全球 LLM（大语言模型）Token 价格查询面板。覆盖 55+ 个模型、30+ 供应商和 9 个地区的数据。

## ✨ 功能特性

- 🌍 **全球覆盖**：55+ 个模型、30+ 供应商、9 个地区
- 💰 **价格对比**：直观的定价数据，支持按模型/供应商筛选
- 📊 **数据可视化**：图表展示价格趋势
- 🔍 **快速搜索**：支持按模型名称、供应商搜索
